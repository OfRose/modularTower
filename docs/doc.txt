ciclo interazione con dispositivi da hub:

1) Scan di tutti gli indirizzi validi:
  Wire library uses 7-bit addressing for a total of 128 addresses from 0x00 to 0x7F
  the usable and not reserved range is between 0x08 and 0x77 -> 112 usable addresses.

  Wire.beginTransmission(address);
  uint8_t end_of_trasmission_byte = Wire.endTransmission();
  end_of_trasmission_byte == 0 ---> dispositivo trovato!

1b) Salvo i dispositivi disponibili trovati in un vettore di stato lungo 112, in
  cui la posizione corrisponde all'indirizzo del dispositivo, aggiornandone lo
  stato in DISCOVERED_TO_INITIALISE:

  enum device_statuses
  {
      NOT_FOUND,                // 0
      DISCOVERED_TO_INITIALISE, // 1
      INSTALLED,                // 2
      ERROR,
  };

2) per ogni dispositivo in stato DISCOVERED_TO_INITIALISE effettuo una requestFrom

  -> ogni dispositivo avrÃ  una mappa di codice_comando e lunghezza_byte (da ricevere o da inviare)
  -> la requestFrom fetcha automaticamente la lunghezza in byte della mappa allo scopo di acquisirla con
      una read dopo aver mandato un comando standard (es: 0x00) per ricevere la mappa dal dispositivo

ES:

MAPPA:
(byte_comando;Nome_funzione;lunghezza_byte In o OUT in base al device;descrizione)

0x01;accendi_led_verde;0;accende led verde
0x02;accendi_led_giallo;0;accende led giallo
